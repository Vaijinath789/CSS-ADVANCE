CSS PSEUDO-ELEMENTS..!

The CSS pseudo -elements allow you to syle the elements or parts of the elements without adding any IDS or classes to them.

CSS # introduced a new double-colon(::) syntax for pseudo -elements to distinguish between them and pseudo-class.

selector::pseudo-element(property:value;)
